![unity安装包](https://upload-images.jianshu.io/upload_images/8399110-b2a3b55d689ab0ad.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

[unity安卓包安装失败](https://img-blog.csdn.net/20180315210402149)
